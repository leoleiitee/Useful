# Fontes úteis para projetos em advpl

Este repositório é destinado a guarda de fontes uteis para reutilização em projetos distintos utilizando ADVPL.

## Lista de fontes:
FONTES     | DESCRIÇÃO
-----------|------------------------------------------------------------------
userErro0  | Funcao que realiza o controle e tratamento de erro em ADVPL
imgenc01   | Funcao que realiza a importação de quelquer CSV para uma tabela do protheus, via reclock ou execauto.
